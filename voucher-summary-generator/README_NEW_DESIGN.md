# 🎨 总台收入工作台 - 中式美学版

基于 ai_studio_code.html 设计风格改造的现代化财务处理系统

## 🚀 快速启动

### 方法一：使用启动脚本
```bash
cd voucher-summary-generator
./run_app.sh
```

### 方法二：直接运行
```bash
cd voucher-summary-generator
streamlit run app.py
```

### 方法三：Python 模块方式
```bash
cd voucher-summary-generator
python3 -m streamlit run app.py
```

访问地址：http://localhost:8501

## 💡 使用建议

### 最佳实践
1. **上传文件**：确保 Excel 包含"工作表"和"总数"工作表
2. **参数设置**：根据实际业务选择正确的收入类型
3. **分配调整**：仔细检查每项分配的准确性
4. **结果验证**：导出前确认所有校验通过

### 故障排除
- 如果页面显示异常，请清除浏览器缓存
- 如需启用在线字体（Google Fonts），检查网络连接或按说明开启 `@import`
- 建议使用 Chrome 或 Safari 浏览器获得最佳体验

## 📞 技术支持

如遇问题，请检查：
1. Python 版本：3.9+
2. 依赖包：pandas, streamlit, openpyxl
3. 网络连接：默认不需要（仅在你启用 Google Fonts 时需要）
4. 浏览器：推荐使用现代浏览器
